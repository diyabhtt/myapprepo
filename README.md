# Member & Claims Intelligence

Prototype member portal built with React, TypeScript, Vite, Tailwind CSS, and local CSV data.

## Run

If `node` is not installed globally, use the repo-local runtime:

```bash
export PATH="$PWD/.local/node/v24.17.0/node-v24.17.0-darwin-arm64/bin:$PATH"
npm install
npm run dev
```
